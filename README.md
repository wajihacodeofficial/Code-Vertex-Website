High-end digital engineering and software development for startups and global enterprises. Specializing in scalable systems, AI, and performance-driven solutions.

## 🚀 Deployment Guide

This project is built with **Vite**, **React**, and **TypeScript**. It is optimized for high performance and is ready for deployment.

### Prerequisites

- Node.js (v18+)
- npm

### 🛠️ Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

### 📦 Production Build

```bash
# Create optimization production bundle
npm run build

# Preview production build locally
npm run preview
```

### 🌐 Deploying to Vercel / Netlify

1. Connect your GitHub repository to Vercel or Netlify.
2. The system should automatically detect the framework (Vite).
3. Set the **Build Command** to: `npm run build`
4. Set the **Output Directory** to: `dist`
5. Add the following Environment Variables if using EmailJS:
   - `VITE_EMAILJS_SERVICE_ID`
   - `VITE_EMAILJS_TEMPLATE_ID`
   - `VITE_EMAILJS_PUBLIC_KEY`
